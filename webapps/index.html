Webアプリケーションの実装¶

Django を使って Web アプリケーションを作成します。

Django は Web アプリケーションのフレームワーク (WAF) で、次の機能を持ちます。

  • O/R マッパー (Object-relational mapper)
  • 管理者向けインターフェイス (Automatic admin interface)
  • きれいな URL 設計 (Elegant URL design)
  • テンプレートシステム (Template system)
  • キャッシュシステム (Cache system)
  • 国際化機構 (Internationalization)
