  • ANS-prog: pythonの-m指定で便利なモジュール類 - freia.jp

JSON の整形¶

ネットワークプログラミング の「HTTP でダウンロード」で作成した downloader.py を使って、 Web サービスから JSON をダウンロードできます。 しかし、Web サービスが出力する JSON は空白や改行などを含まないため、人が読むには適していません。 いくつかのビューアーもありますが、Python のモジュールを利用して整形することもできます。

json.tool というモジュールは実行可能なモジュールになっています。 Python インタープリタの -m オプションに指定することで、 モジュールを実行できます。

JSON のダウンロード (Twitter で「Python」という単語を検索)

$ python downloader.py "http://search.twitter.com/search.json?q=Python"
INFO:root:Download: http://search.twitter.com/search.json?q=Python -> search.json

JSON の整形

$ python -m json.tool search.json >search-pretty.json

整形結果の確認

HTTP サーバの起動¶

Python 2 系では SimpleHTTPServer というモジュールを実行すると、 カレントディレクトリを HTTP サーバで公開できます。

> python -m SimpleHTTPServer

Python 3 系では http.server に変更されています。

> python3 -m http.server

アドホックなネットワーク環境でファイルを交換する場合や、 JavaScript を実装する場合には便利なこともありますので、覚えておいてください。

日本語を扱うときに便利なこと¶

JSON に含まれる日本語をエスケープさせない¶

  • python2.6でdictionaryをjson化した際、日本語がuXXXXにエスケープされないようにする

json-dump.py

# -*- coding: utf-8 -*-

import codecs
import json

data = {"results":[
  {
    "name": u"国立霞ヶ丘競技場",
    "latitude": 139.714941,
    "longitude": 35.678160,
    "description": "naash.go.jp"
  },
  {
    "name": u"味の素スタジアム",
    "latitude": 139.5272,
    "longitude": 35.6646,
    "description": "http://www.ajinomotostadium.com/\n"
                   u"元の名称: 東京スタジアム"
  }
]}
with codecs.open('out.json','w','utf-8') as writer:
    json.dump(data, writer, indent=4, ensure_ascii=False)
print "Wrote to \"out.json\"."
